# Weapon Images

To use the dynamic weapon image feature, save the following images in this folder:

## Required Images:
1. **shotgun.jpg** - Green holographic shotgun image (first image you provided)
2. **pistol.jpg** - Green holographic pistol image (second image you provided)  
3. **sniper.jpg** - Green holographic sniper rifle image (third image you provided)
4. **rifle.jpg** - Green holographic rifle image (fourth image you provided)

## How it works:
- When you select "Shotgun" filter → shows shotgun.jpg
- When you select "Pistol" filter → shows pistol.jpg
- When you select "Sniper" filter → shows sniper.jpg  
- When you select "Rifle" filter → shows rifle.jpg

## Fallback:
If the local images are not found, the system will use online placeholder images.

## Image Requirements:
- Format: JPG or PNG
- Recommended size: 400x300 pixels
- Style: Green holographic/neon effect on dark background
